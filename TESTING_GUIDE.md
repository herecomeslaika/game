## 🐛 常见问题排查

### Q: 按钮没有显示发光效果
**排查**：
- [ ] 检查GPU是否支持Paint阴影
- [ ] 查看ANTI_ALIAS_FLAG是否正确设置
- [ ] 检查Color.argb()的透明度值

### Q: 加载动画不转动
**排查**：
- [ ] 检查`renderer.globalTime`是否在更新
- [ ] 验证`globalTime`的更新频率（应该是60fps）
- [ ] 检查canvas.rotate()的计算是否正确

### Q: 文字显示不清晰
**排查**：
- [ ] 检查字体大小是否过小
- [ ] 验证文字颜色与背景的对比度
- [ ] 检查setShadowLayer()的参数

### Q: 玩家列表显示不全
**排查**：
- [ ] 检查`if (y > h * 0.75f) break`的边界条件
- [ ] 验证itemSpacing和itemHeight的值
- [ ] 检查列表起始Y坐标

## 📊 性能考虑

### 优化建议
1. **网格背景优化**：可缓存为Bitmap而不是每帧重绘
2. **加载动画优化**：可使用ObjectAnimator替代手动计算
3. **多个Paint对象**：已优化，减少GC压力

### 预期性能
- 渲染帧率：60 FPS（正常设备）
- 内存使用：< 5MB（UI部分）
- CPU占用：< 10%（UI渲染）

## 💡 调试技巧

### 查看UI边界
在 `drawPlayerItemCard()` 中添加：
```kotlin
// 调试：显示边界框
renderer.paint.color = Color.argb(100, 255, 0, 0)
renderer.paint.style = Paint.Style.STROKE
canvas.drawRect(x, y, x + width, y + height, renderer.paint)
```

### 测试触摸区域
在 `handleTouch()` 中添加日志：
```kotlin
Log.d("UI_Debug", "Touch at ($x, $y)")
if (readyBtnRect.contains(x, y)) {
    Log.d("UI_Debug", "Ready button touched!")
}
```

### 帧率监测
添加FPS计数器：
```kotlin
private var frameCount = 0
private var lastTime = System.currentTimeMillis()

fun updateFPS() {
    frameCount++
    val now = System.currentTimeMillis()
    if (now - lastTime >= 1000) {
        Log.d("FPS", "Current FPS: $frameCount")
        frameCount = 0
        lastTime = now
    }
}
```
